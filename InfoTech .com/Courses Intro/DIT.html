DIT

What Is DIT:

DIT stands for Diploma in Information Technology is 1 year diploma program offered by Board of Technical Education (BTE) Peshawar having two semesters. The purpose of this program to be able student in practical training such as:

  • Use of computer with various operating system.
  • How to make a computer network?
  • How to use Microsoft Office?
  • How to do desing a flyer or advertisement?
  • How to do computer programming? which will helpful in advance learning?
  • How to make a webpage?

Course Includes:

SEMESTER-1
  1. Information & Communication Technology (ICT)
  2. Computer Programs (C/C++)
  3. Operating System (Windows, Linux)
  4. Computer Network
  5. Office Automation (Word, Excel, Powerpoint)

SEMESTER-2
  1. Introduction to Database (Oracle)
  2. Microsoft Access
  3. E-Commerce & Web Technology (HTML, CSS, JavaScript)
  4. Graphic Design (Photoshop, Corel Draw, Inpage)
  5. Project
